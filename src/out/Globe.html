Class: Globe

Globe(elementId, geocoder)

La classe Globeva permettre de manipuler Cesium plus facilement ; elle contient toutes les fonctions techniques, qui vont permettre d’ajouter des couches de données, de dessiner, de découper, d’ajouter des points de vues de caméra, etc.
Les fonctions peuvent être réutilisées dans un autre projet

Constructor

new Globe(elementId, geocoder)

Le constructeur de la classe globe, qui créé :
- le viewer Cesium
- le handler c’est à dire l’objet qui va contenir les évènements liés à la souris
- le tableau contenant les dataSources
- la grille Raf09 pour les conversions hauteur ellipsoïdale/altitude
- le PinBuilder
- déclare toutes les checkbox de la boîte à outils pour l'affichage dynamique
on supprime aussi le fond de plan, on définit la couleur de fond et on ajoute les logos en bas de l’écran
Parameters:
Name Type Description
elementId String Le nom du contenant html dans lequel on l'ajoute
geocoder Object Le Geocoder à associer

setHome()

Elle permet de paramétrer la vue de base de l’application ainsi que le zoom par défaut sur lequel le viewer se repositionne lorsqu’on clique sur le bouton « maison »
Elle ne prend pas de paramètres mais considère deux cas différents :

updateShape(choice, choice2, largeur, couleur, transparence, hauteurVol, url, billboard, line, surface, volume, dline, dsurface)

La fonction qui permet de tout dessiner
le paramètre choice designe si on mesure (dessins temporaires) ou si on dessine (dessins qui restent après fermeture de la fonction de dessin)
le paramètre choice2 désigne le type de dessin (line, surface, volume etc)
On met tous les tableaux d'entités en paramètres de la fonction car ils seront définis dans la classe Menu pour garder une trace des entités et permettre leur annulation/exportation
Le reste des paramètres correspond aux paramètres de personnalisation définis par l'utilisateur dans les formulaires
Parameters:
Name Type Description
choice String prend la valeur 'dessin' ou 'mesure'
choice2 String le type d'entités à dessiner: 'point', 'line', 'polygon' ou 'volume'
largeur Number la transparence de l'entité
couleur String le couleur de l'entité
transparence Number la transparence de l'entité
hauteurVol Number la hauteur de l'entité
url String le lien vers les images pour les entités billboard
billboard Array le tableau où stocker les entités billboard
line Array le tableau où stocker les entités lignes
surface Array le tableau où stocker les entités surface
volume Array le tableau où stocker les entités boîte
dline Array le tableau où stocker les entités lignes pour les mesures
dsurface Array tableau où stocker les entités surface pour les mesures
